VIEWING PARTY

One of the first projects assigned at Ada, Viewing Party required us to create an app that used dictionaries, lists, nested loops, nested data structures, and conditional logic to compare a user's and their friend's watch lists.

SWAP MEET

Swamp Meet was a solo project completed at Ada where "vendors" can swap items between inventories. We had to utilize our limited knowledge of Object Oriented Programming and reading tests to build this app.

TASK LIST API

For Task List API, knowledge of tests, the client-server model, external APIs, Postman, OOP, and RESTful CRUD were required to complete an app that allowed us to create, read, update, connect, and delete tasks and goals.
