Swap-Meet

I wrote Python code to help vendors organize and sell their items. This project includes features that allow you to easily see specific categories of products in their inventory. There is also a feature that allows vendors to swap items based on what items are available in this inventory.

Task-List API

Designed an API using Python to help users organize their to-do lists. I designed various endpoints that allow users to update, delete, and read a task. In addition, I set up a feature that notifies users when their task is complete through a slack bot.

Solar System API

Designed a Solar Systems API to capture information about the planets and their moons. My partner and I designed several endpoints that allowed users to post information about a planet, read information about a specific planet, as well as update and delete planet information.

AdaGrams

My teammate and I created our version of an Anagram game using Python code. In this game, a player is given a random set of letters and must make an anagram using those letters. Each submitted word will score points. This game uses just the English alphabet.
