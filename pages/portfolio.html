This personal portfolio will highlight the projects that I have completed so far as a developer.

Viewing Party

This project required writing functions to handle data about movies and user recommendations. These functions modified the data, and implemented features like adding and removing things between different lists.

Skills assessed:

  • Working with nested lists and dictionaries
  • Writing tests
  • Writing functions
  • Using software tools

Adagrams

In this project a pretend game was created in which a player is given a random set of letters and must make an anagram using those letters. Each submitted word scored points and the highest scored word is calculated.

Skills assessed:

  • Reading tests
  • Manipulating and processing data in lists and strings
  • Using git to maintain code

Swap Meet

This project required writing logic for an app that kept track of vendors and their inventories. The items could be broken down by their categories and swapped between vendors.

Skills assessed:

  • Creating classes and working with instances
  • Importing modules
  • Implementing instance methods
  • Implementing inheritance
  • Overriding methods from superclasses
  • Reading and writing tests

Tasklist API

This project involved creating a web API that could be used to organize tasks and goals
for the user. This project showcases my API skills and my ability to work with the
Flask package.

Skills assessed:

  • Demonstrating understanding of the client-server model, request-response cycle
    and conventional RESTful routes.
  • Reading and using existing external web APIs.
  • Using Postman as part of the development workflow.
  • Creating models with Flask.
  • Creating conventional RESTful CRUD routes for a model.
  • Create non conventional routes for custom behavior.
  • Apply knowledge about making requests in Python, to call an API inside of an API.
