Portfolio

Viewing Party

This was the first project I did during the class portion of Ada. It allows you to keep up with what movies you want to watch or have watched and compares it to your friends watched list to create a recommendation list of movies to watch. I learned about list, dictionaries, nested loops, nested data structures, and conditional logic during this project.

Adagrams

Adagrams the first group project I did at Ada. This project created a game similar to scrabble or bananagrams. I learned about reading test, using git to maintain code, and pair-programming techniques during this project.

Swap Meet

This was another group project at Ada. This project creats a way for vendors to add items to thier inventory and swap items with other vendors. I learned how to work with classes and methods, how to implement inheritance, about object oriented programming in general during this project.

Solar System API

This was a group project used to introduce us to flask. We created an database of the planets. Then we created an api that used RESTful CRUD operations to maintain that database. This project taught me how to use Flask, how to use SQLAchemy, and allowed me to srengthen my pair-programming skills.

Task List API

This was a solo project used to cement my understanding of Flask and SQLAchemy. It created a web api that would keep track of goals and the task required to complete those goals. With this project i learned how to write test for flask apps, how to use Postman, how to read and use existing external web APIs, and how to deploy my own API.
