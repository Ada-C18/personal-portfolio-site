Since becoming a student at ADA developers Academy, Sunny has the opportunity to
work on and complete various projects either with a partner or alone. Below is a short
description of Sunny's projects as well as the Github repository links.

Solo Projects

  • Snowbug

    • Snowbug is a game in which a player guesses
    the letter in an uknown word. Every time the
    user guesses a wrong letter, they slowly build
    a snowman, until they run out of guesses.

  • Viewing Party

    • Viewing Party is a program that lets users collect
    modify, and organize movie data, such as: watched movie lists, favorite movie lists, friends watched
    movie lists, and more!

  • , Task List API

    • Task List is a web API that helps a user
    stay organized by making it easy to create
    read, update, and delete tasks among other
    features.

Pair Projects

  • Adagrams

    • Sunny worked on this project with Marie Keefer.
    Adagrams is a fun game in which a player is given
    a set of random letters and has to use those letters
    to create a word

  • Swap Meet

    • Sunny worked on this project with Samiya Ismail.
    Swap Meet is a program that helps users organize
    swap meets by keeping track of a vendors inventory
    wanted items, and swapped items among other features.

  • Solar System API

    • Sunny worked on this project with Kat Hawkins.
    Solar System API is an API that lets users store,
    update, delete, create, and request information on
    different planets.
