Portfolio

Task List API

An API that allows users to create, read, update, and delete tasks and goals. When a task is completed, a message will be posted to a #task-notifications channel on Slack to notify all members of finished tasks via Slack Bot.

Solar System API

An API that allows users to create, read, update, and delete planets. Each planet has a name, id, description, and number of moons. A user can search for one planet by id or name, or even display all planets at once.

Swap Meet

This App makes bargaining easy! A vendor can swap items based on category and condition to find the perfect item quickly. It even considers other vendor's priorities so that both vendors swap for only the items they truly want!

ADAGRAMS

ADAGRAMS is a fun game that gives the player a random set of letters from the English alphabet with which they must create an anagram. Each letter has an associated point value, which this game recognizes and keeps score of.

Viewing Party

Viewing Party is a project that compares your watched, favorited, and not-yet-watched movies with your friends' watched, favorited, and not-yet-watched movies. Spend less time endlessly scrolling for a movie to watch and more time watching!
