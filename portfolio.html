Viewing Party

Viewing Party is a project that helps friends connect over movies by tracking and organzing watchlists while also making new movie suggestions!

Ada-Grams

Adagrams, a game in which a player is given a random set of letters and must make an anagram using those letters. Each submitted word will score points.

Swap Meet

Swap Meet allows people at a swap meet to swap items between different inventories. Vendors are be able to swap their top item and swap items by category!

Task List API

Task List API organizes our goals into smaller, bite-sized tasks, we'll be able to track them more easily, and complete them!
